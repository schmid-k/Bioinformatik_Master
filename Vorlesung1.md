# Vorlesung 1 (20.11.)

## Python Grundlagen

### Datentypen

- Integer
```python
variable1 = 1
variable2 = variable1 + 2
```

- Floats
```python
variable1 = 1.00
variable2 = variable1 + 3.14
```

- String
```python
variable1 = "Text"
variable2 = variable1 + "input"
```

- Boolean
```python
variable1 = True
variable2 = False
```
